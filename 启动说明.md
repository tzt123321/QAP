# 启动说明

## 前置要求

1. **Java 17+** - 用于运行SpringBoot后端
2. **Maven 3.6+** - 用于构建后端项目
3. **Node.js 16+** - 用于运行Vue3前端
4. **MySQL 8.0+** - 数据库

## 快速启动步骤

### 1. 数据库准备

```sql
-- 创建数据库
CREATE DATABASE quick_access DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

-- 或者直接执行初始化脚本
mysql -u root -p < database/init.sql
```

### 2. 配置数据库连接

编辑 `backend/src/main/resources/application.yml`，修改数据库连接信息：

```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/quick_access?useUnicode=true&characterEncoding=utf8&useSSL=false&serverTimezone=Asia/Shanghai
    username: root
    password: 你的密码
```

### 3. 启动后端

```bash
cd backend
mvn clean install
mvn spring-boot:run
```

后端将在 `http://0.0.0.0:8999` 启动

### 4. 启动前端

打开新的终端窗口：

```bash
cd frontend
npm install
npm run dev
```

前端开发服务器将在 `http://localhost:3000` 启动

## 访问方式

### 普通访问模式
- 访问：`http://你的IP:8999` 或 `http://localhost:3000`
- 功能：查看应用列表，点击卡片跳转

### 管理模式
- 访问：`http://你的IP:8999?key=123456` 或 `http://localhost:3000?key=123456`
- 功能：增删改查Web应用
- 默认密匙：`123456`（可在 `application.yml` 中修改）

## 生产环境部署

### 后端
```bash
cd backend
mvn clean package
java -jar target/quick-access-backend-1.0.0.jar
```

### 前端
```bash
cd frontend
npm run build
# 将 dist 目录内容部署到静态服务器或后端静态资源目录
```

## 常见问题

1. **端口被占用**：修改 `application.yml` 中的 `server.port`
2. **数据库连接失败**：检查MySQL服务是否启动，用户名密码是否正确
3. **前端无法连接后端**：检查后端是否启动，端口是否正确
